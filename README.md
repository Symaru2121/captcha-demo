# CAPTCHA Sandbox v6 — Demo

5 кастомных капч для платформы Sandbox v3.

## Капчи

| # | Название | Механика |
|---|----------|----------|
| 1 | Найди лишнее | Выбери лишний предмет из 6 эмодзи |
| 2 | Что невозможно? | Найди абсурдное утверждение из 4 |
| 3 | Нерусские буквы | Отметь все не-кириллические символы |
| 4 | Поверни правильно | Поверни силуэт в правильную ориентацию |
| 5 | Собери кости | Найди карточку с нужной суммой костей |

## Технологии

- Canvas-only рендеринг (защита от ботов)
- Proof-of-Work модуль
- Поведенческий анализ (trajectory, velocity, timing)
- Бинарный протокол (компактная передача данных)
- Комбинатор капч (случайный выбор 1/5)

## Демо

Откройте `index.html` — лаунчер со всеми 5 капчами.

Или отдельные файлы:
- `captcha1_demo.html` — Найди лишнее
- `captcha2_demo.html` — Что невозможно?
- `captcha3_demo.html` — Нерусские буквы
- `captcha4_demo.html` — Поверни правильно
- `captcha5_demo.html` — Собери кости

Работает как `file://`, сервер не нужен.
