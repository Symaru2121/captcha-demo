### Что внутри

- **Найди лишнее** — сетка эмодзи, 24 категории, Canvas
- **Что невозможно в реальной жизни?** — 96 наборов заданий, текст на Canvas
- **Найди нерусские буквы** — мультивыбор, 3 уровня сложности

### Дополнительно

- Комбинатор капч (случайный выбор типа)
- Поведенческий анализ (трекинг мыши, скоринг человек/бот)
- Proof-of-Work модуль
- Go-бэкенд под SDK Sandbox v3
- Документация на русском
